## qt qml样式

工作关系会经常使用qt，有事没事的时候，就会做一些qt的组件/效果，将其保存于此。(＞﹏＜)

#### 使用方法

clone下来后，用qt creator打开qmlStyle.pro文件。然后run一下，就会看到所有可显示控件的使用示例，对哪个有兴趣，可以详细查看其代码。
