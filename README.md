#### Qt 版本说明

目前工作环境为 Qt Creator 3.6.1 Based on Qt 5.6.0.

如果出现打开此项目各种不能编译、爆红等错误，请考虑是否是 Qt 版本过低导致。

#### 使用方法

clone 下来后，用 qt creator 打开 qmlStyle.pro 文件。然后 run 一下，就会看到所有可显示控件的使用示例，对哪个有兴趣，可以详细查看其代码。
